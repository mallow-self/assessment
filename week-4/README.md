# Todo - Instructions
1. Create a conda venv here:
    - `conda create -n todo_env python=3.12`
2. Activate it
3. Initialize poetry for the project:
    - `poetry install`

4. Go into todo:
    - `python3 manage.py makemigrations`
    - `python3 manage.py migrate`
    - `python3 manage.py runserver`
